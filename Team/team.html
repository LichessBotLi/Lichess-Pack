Lichess Package Project

Team Messages

Usage

The Lichess Package Project gives user options to send lichess team message any time ,any where he/she wants. Many a time it happens you need to send an important team message.However you are not free at that time to sent it.Lichess Package makes it easier to set your team message 12 hours before.

Tutorial

Well here comes the main part,how do you use it? It is much simpler than it seems.Just below click the authorize button below. You can unauthorize any time you want.Once done just set your time to send the message according to GMT.Then enter your team id(for example lp-eu if team link is lichess.org/team/lp-eu).Enter the message to be sent and start.You are done now.Once the message is sent Lichess Package will automatically unauthorize your account since we do not want any risk:D
